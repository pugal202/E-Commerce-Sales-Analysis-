# E-Commerce Sales Analysis

## Project Objective

Analyze e-commerce sales data to identify revenue trends, top-performing product categories, regional performance, customer review patterns, and business opportunities.

## Tools Used

* Microsoft Excel
* Power Query
* Pivot Tables
* Pivot Charts

## Key Findings

* Highest Revenue Category: bed_bath_table
* Peak Sales Month: May
* Best Performing State: SP
* High Customer Satisfaction through 5-star reviews
